# vanity
Generating vanity urls for TOR nodes and I2P EEPSITES

```
vanityctl stop i2p; vanityctl empty i2p; vanityctl add i2p "^aochange[0-9].*$" "^change[0-9].*$" "^njtsec[0-9].*$" "^njphoto[0-9].*$" "^darknige[0-9].*$" "^drdeviant[0-9].*$"; vanityctl start i2p
```
